# Falaah - Pakistan NGO Directory

**Search. Discover. Connect.**

Falaah is a free, publicly accessible web portal that brings Pakistan's social welfare landscape into one searchable, categorised directory. Built as a community service initiative in collaboration with [Shah Qutub Welfare Foundation (SQWF)](https://sqw.foundation), it connects citizens, donors and volunteers with organisations doing meaningful work across the country.

**Live site: [falaah.vercel.app](https://falaah.vercel.app)**

---

## What It Does

- Lists 250+ NGOs across 10 categories of social work in Pakistan
- Search by name, category, focal person or location
- Filter by province
- Public NGO submission form with admin review workflow
- Contact form with messages saved to database
- Admin data management via Supabase dashboard
- Fully responsive across mobile, tablet and desktop

## Database Schema

Two primary tables in Supabase:

**ngos** - All listed organisations
- UUID primary key, auto-generated
- Foreign key to categories table
- Slug auto-generated by database trigger
- Row Level Security: public can read active entries, public can submit pending entries, admins have full access

**categories** - The 10 social work categories
- Reference table with display order
- Joined to ngos on every public query

**contact_messages** - Messages from the contact form
- Name, email, subject, message
- Public insert, admin read

## Running Locally

No build tools or package managers required.

1. Clone the repository
2. Create a `config.js` file in the root directory:

```javascript
window.FALAAH_CONFIG = {
    SUPABASE_URL: 'your-supabase-project-url',
    ANON_KEY: 'your-supabase-anon-key'
};
```

3. Open `index.html` in your browser

For local development with live reload, use the VS Code Live Server extension.

---

## Deployment

The site is deployed on Vercel connected to a GitHub repository. Every push to the main branch triggers an automatic redeployment within 30 seconds.

Database is hosted on Supabase with Row Level Security enforced. The public anon key is scoped by RLS policies and cannot access any data beyond what is intended for public display.

---

## Security

- All traffic encrypted via TLS (enforced by Vercel)
- Database encrypted at rest via Supabase (AES-256)
- Row Level Security policies restrict all public database access
- Service role key is never present in frontend code
- Full SOC 2 readiness assessment conducted covering all five Trust Service Criteria

## About This Project

Falaah was built as part of a community service project with Shah Qutub Welfare Foundation. The goal was to solve a real problem: Pakistan has thousands of active NGOs but no single, clean, searchable place for the public to find them.

The project was scoped, designed, built and delivered in four weeks by a team of five, with full handover documentation, a data management guide and a security report provided to the client organisation.
